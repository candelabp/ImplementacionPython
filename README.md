# Perceptrón Modificado

Siguiendo los requisitos de la cátedra, se realizó una modificación funcional al código base del Perceptrón. Originalmente diseñado para resolver la lógica AND, el sistema fue extendido para soportar también la operación OR.

## Detalles de la modificación
Modelo: Se ajustaron los pesos (w) y el umbral de activación (bias) para permitir que el perceptrón clasifique correctamente ambas tablas de verdad.

Controlador: Se añadió lógica para que el usuario pueda elegir qué operación lógica desea evaluar antes de ejecutar el entrenamiento/predicción.

Vista: Se actualizó la interfaz de consola para mostrar los resultados específicos de la compuerta OR seleccionada.
